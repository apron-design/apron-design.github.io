输入框 Input

通过鼠标或键盘输入内容，是最基础的表单域包装。

何时使用

  • 需要用户输入表单域内容时
  • 提供组合型输入框，比如带前后置内容的输入框
  • 需要带清除功能的输入框时

参数说明类型默认值

clearable是否显示清除按钮booleanfalse
