标签页 Tabs

Tabs 组件用于组织和展示不同类别的内容，允许用户在不同的视图之间进行切换。

基础用法

Tabs 组件由四个部分组成：

  • Tabs: 容器组件
  • TabList: 标签列表容器
  • Tab: 单个标签项
  • TabPanel: 标签对应的内容面板
