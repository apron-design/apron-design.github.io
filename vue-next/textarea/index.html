API

ad-textarea

参数说明类型默认值
rows默认行数number3
clearable是否显示清除按钮booleanfalse
on-clear清除时的回调() => void-
disabled是否禁用booleanfalse
max最大字数限制，设置后显示字数计数number-
value输入框内容（受控）string-
default-value输入框默认内容string-
on-change输入框内容变化时的回调(event: Event) => void-
on-focus输入框获得焦点时的回调(event: FocusEvent) => void-
on-blur输入框失去焦点时的回调(event: FocusEvent) => void-
class-name自定义类名string-
placeholder占位文本string-
