标签页 Tabs

Tabs 组件用于组织和展示不同类别的内容，允许用户在不同的视图之间进行切换。

基础用法

Tabs 组件由四个部分组成：

  • ad-tabs: 容器组件
  • ad-tab-list: 标签列表容器
  • ad-tab: 单个标签项
  • ad-tab-panel: 标签对应的内容面板

API

参数说明类型默认值

defaultActiveKey默认激活的标签页string-
